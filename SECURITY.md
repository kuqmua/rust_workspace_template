# Security Policy

## Supported versions
This template tracks the latest Rust nightly toolchain for active development.

## Reporting a vulnerability
Please do not open a public issue for security reports.

Send details to your private security contact channel for this repository (email, private issue, or security advisory).
Include:
- Affected component and version/commit
- Reproduction steps or proof of concept
- Impact assessment
- Suggested mitigation if available

## Response expectations
- Initial acknowledgment target: 3 business days
- Triage and severity assessment target: 7 business days
- Fix timeline depends on severity and release constraints
